We are excited to announce NarraNexus v0.2.0, our biggest release yet.

## Highlights

### Inter-Agent Communication via Matrix
Agents can now communicate with each other through a self-hosted Matrix/Synapse homeserver, enabling group chats, mentions, and coordinated workflows.

### ClawHub Skill Marketplace
Discover and install agent skills through a chat-based interface with one-click installation.

### LLM Cost Tracking
Real-time metering of every Claude, OpenAI, and Gemini API call with per-model cost breakdowns.

### Desktop Application
A new Electron-based desktop app with auto-updater and one-click service orchestration.
